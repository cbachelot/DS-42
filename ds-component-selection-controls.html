Selection Controls

Selection controls allow users to complete tasks that involve making choices such as selecting options, or switching settings on or off. Selection controls are found on screens that ask users to make decisions or declare preferences such as settings or dialogs.

Selection Control Types

The most common selection controls types are:

Checkboxes

This pattern allows the user to make multiple selections from a list of items. It can also be paired with a parent checkbox for easy of selecting and deselecting multiple checkboxes.

Parent and child checkboxes
Checkboxes can have a parent and child relationship. For example:

  • When the user selects a parent checkbox then all child checkboxes below are checked.
  • So when a user unselects a parent checkbox then all child checkboxes are unchecked
  • If only some of the checkboxes are selected but not all then the parent checkbox becomes an indeterminate checkbox.

Radio Buttons

Radio buttons allow a user to select one option from a list of items. It's a good idea to use radio buttons when the user needs to see all available options. If the user doesn't need to see all the options or the amount of options becomes overwhelming then a dropdown would be a good alternative because it saves space.

Toggles

Toggles switch the state of a setting or feature on or off. When a user toggles something on or off the result should take effect immediately.
