Date Pickers

Date pickers allow the user to input a date or a range of dates into our UI. An example of a single date picker is setting a specific shipping schedule date in Connect or in checkout. An example of using a date range would be during filtering of results by setting a specific date range.

Date Picker Types

The most common date picker types are:

Mobile Date Picker

Mobile date pickers are used to select future or past dates on a mobile or tablet device. The user can swipe up and down to change months for ease of use while the days of the week and apply button will stay stationary.

Desktop Date Picker

Desktop date pickers are used to select a date or a range of dates into our desktop UI. On desktop we are able to show an expanded two pane view for quicker selection. Months can be changed by using the chevrons to the left and right.

Calendar Anatomy

1. Text field - See form field documentation for specs.
2. Month and year – 12px Verdana Regular font - Color: #2c2c2c
3. Day of the week- 10px Verdana Bold font - Color: #2c2c2c
4. Disabled dates - Color: #979797
5. Current date – Background color: #f4f4f4
6. Navigation chevrons – Active color: #979797 - Disabled color: #f4f4f4
7. Hover state – 1px outline: #008BAA
8. Selected date – Background color: #008BAA
9. Clear button – Clears out the selected date
10. Apply button – Will apply the date to the UI and close selection modal
